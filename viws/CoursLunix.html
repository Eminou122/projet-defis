Leçon 3 • Permissions & sudo

Les permissions : qui peut faire quoi ?

Tout fichier/dossier a des droits : lire (r), écrire (w), exécuter (x)

$ ls -l
-rw-r--r-- 1 akon akon 1240 avril 5 rapport.txt

Explication :

  • -rw- → toi tu peux lire + écrire
  • r-- → les autres peuvent seulement lire
  • r-- → le groupe aussi

Changer les permissions → chmod

Tu veux rendre un script exécutable ?

$ chmod +x mon-script.sh
→ Maintenant tu peux lancer ./mon-script.sh

Tu as téléchargé un programme ? chmod +x et c’est bon !

Devenir administrateur → sudo

sudo = Super User DO → tu deviens temporairement le patron

$ sudo apt update
[sudo] Mot de passe de akon :

Tu veux installer un logiciel ? sudo te donne le pouvoir !

Attention : avec sudo tu peux tout casser → utilise-le seulement quand c’est nécessaire

Changer le propriétaire → chown

Donner un fichier à quelqu’un d’autre :

$ sudo chown mariam rapport.pdf

Tu travailles en équipe ? chown donne le fichier à ton collègue !

Récapitulatif des commandes magiques

$ ls -l → voir les permissions
$ chmod +x fichier → rendre exécutable
$ sudo commande → avoir tous les pouvoirs
$ sudo chown user fichier → changer le propriétaire

Leçon 2 • Gestion des fichiers

Créer un dossier → mkdir

Pour créer un nouveau dossier :

$ mkdir photos-vacances
→ Un dossier "photos-vacances" est créé

Tu reviens de voyage ? Crée un dossier en 1 seconde !

Copier un fichier → cp

Copier un fichier ou un dossier :

$ cp photo.jpg photo-copie.jpg
$ cp -r photos-vacances photos-vacances-sauvegarde

Tu veux garder une sauvegarde ? cp fait tout pour toi !

-r = récursif (pour copier tout un dossier avec son contenu)

Déplacer / Renommer → mv

Pour déplacer ou renommer :

$ mv photo.jpg Images/
$ mv ancien-nom.jpg nouveau-nom.jpg

Tu as mal nommé un fichier ? mv le renomme instantanément !

Supprimer → rm

Attention : rm supprime définitivement !

$ rm fichier-inutile.txt
$ rm -r dossier-inutile
Pas de corbeille sous Linux → sois sûr avant de taper rm !

Voir le contenu d’un fichier → cat

Afficher tout le contenu d’un fichier texte :

$ cat recette-pates.txt
Ingrédients : pâtes, sauce tomate, fromage...

Tu as un fichier texte ? cat te le lit en entier !

Créer un fichier vide → touch

Créer un fichier vide ou mettre à jour la date :

$ touch ma-liste-courses.txt

Tu veux commencer une nouvelle liste ? touch crée le fichier en 0 seconde !

Leçon 1 • Bienvenue dans Linux !

Qu'est-ce que Linux ?

Linux est un système d’exploitation libre et gratuit, créé en 1991 par Linus Torvalds.

Contrairement à Windows ou macOS, tu peux le modifier, le distribuer et l’utiliser comme tu veux !

Aujourd’hui, Android, les serveurs web, les superordinateurs et même les routeurs tournent sous Linux !

Le Terminal : ton nouveau meilleur ami

Dans Linux, on ne clique pas partout : on tape des commandes dans le terminal. C’est plus rapide et plus puissant !

Exemple : au lieu de chercher un dossier pendant 2 minutes avec la souris → 1 commande = 1 seconde.

Commande #1 → uname -r

Afficher la version du noyau Linux

$ uname -r
6.5.0-28-generic

Exemple simple : Tu veux savoir si ton ordi est à jour ? Tape cette commande → tu vois tout de suite la version !

Commande #2 → whoami

Voir ton nom d’utilisateur actuel

$ whoami
akon

Tu es sur un ordi partagé ? Tape whoami → tu sais tout de suite si c’est bien ton compte !

Commande #3 → pwd

Afficher le dossier où tu te trouves (pwd = Print Working Directory)

$ pwd
/home/akon/Bureau

Tu es perdu dans les dossiers ? Tape pwd → tu sais exactement où tu es !

Commande #4 → ls

Lister les fichiers et dossiers

$ ls
Documents Images Téléchargements musique
$ ls -la
drwxr-xr-x 4 akon akon 4096 avril 5 Documents
-rw-r--r-- 1 akon akon 12345 mars 30 photo.jpg

Tu cherches un fichier ? ls te montre tout en 1 seconde !

Commande #5 → cd

Changer de dossier (cd = Change Directory)

$ cd Documents
$ cd ..
$ cd ~

Tu veux aller dans ton dossier Téléchargements ? → cd ~/Téléchargements
